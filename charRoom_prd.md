# 项目目标

实现一个基于TCP协议的多人在线终端聊天室

------------------------------------------------

# 项目范围

客户端与服务端均运行于Linux

使用C++开发

采用TCP通信

不支持图形化界面,支持终端聊天

仅维护一个聊天室

聊天室内用户昵称唯一

------------------------------------------------

# 功能需求

用户:

1. 登录聊天室

2. 发送聊天消息

3. 接收其他用户消息

4. 退出聊天室

客户端:

1. 建立TCP连接

2. 接收用户输入

3. 发送消息到服务器

4. 接收服务器转发消息

服务端:

1. 监听客户端连接

2. 管理在线用户

3. 处理用户上线和下线

4. 接收客户端消息

5. 广播客户端消息

------------------------------------------------

# 消息类型

LOGIN  用户上线

CHAT   用户聊天消息

LOGOUT 用户退出聊天室

------------------------------------------------

# 非功能需求

Linux运行

TCP通信

支持10个用户同时在线

单条消息最大1024字节

------------------------------------------------

# 验收标准

支持3个以上客户端同时连接

客户端发送消息能够被其他客户端接收

客户端退出后服务器正确释放资源
